Рингтоны

Авторские рингтоны от Mednov. Слушайте, скачивайте, устанавливайте.

Как установить рингтон на iPhone
  1. Скачайте файл .m4a на iPhone
  2. Откройте GarageBand (бесплатно в App Store)
  3. Создайте проект → Аудиорекордер
  4. Иконка петли (вверху справа) → Файлы → выберите файл
  5. Перетащите на дорожку
  6. Долгое нажатие на проект → Поделиться → Рингтон
  7. Настройки → Звуки → Рингтон — выберите
Как установить рингтон на Android
  1. Скачайте файл .mp3
  2. Настройки → Звуки → Мелодия звонка
  3. Нажмите «+» / «Добавить»
  4. Выберите файл из Загрузок

Или скопируйте .mp3 в папку Ringtones через файловый менеджер.

Как установить звук в Telegram
  1. Скачайте файл с пометкой tgm (до 5 сек)
  2. Telegram → Настройки → Уведомления и звуки
  3. Звук уведомлений → Загрузить звук
  4. Выберите файл — готово!
