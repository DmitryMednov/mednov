# 🚢 Морской Бой · Онлайн

Сетевая версия классического морского боя на два игрока. Node.js + Socket.IO.

## Как играть

1. Игрок вводит ник
2. **Создаёт комнату** → получает код (напр. `K7N3F`)
3. Отправляет код другу
4. Друг вводит код и подключается
5. Оба расставляют флот (перетаскиванием или случайно)
6. Нажимают "Готов к бою" → игра начинается
7. Ходят по очереди. Попал — стреляешь снова, промахнулся — ход противника
8. Кто первый потопил весь флот противника — победа

---

## 🚀 Локальный запуск (для проверки)

```bash
npm install
npm start
```

Открой `http://localhost:3000` в двух окнах браузера — одно создаёт комнату, второе подключается по коду.

## 🌐 Деплой на Render.com (бесплатно)

### Шаг 1. Залить код на GitHub

1. Зарегистрируйся на github.com, если ещё нет
2. Создай новый репозиторий (например `battleship-online`), публичный
3. Загрузи в него **все файлы из этой папки**:
   - `server.js`
   - `package.json`
   - `public/index.html`
   - `README.md`

Проще всего через кнопку "Add file → Upload files" прямо на сайте GitHub.

### Шаг 2. Создать сервис на Render

1. Зарегистрируйся на [render.com](https://render.com) (бесплатно, можно через GitHub)
2. Нажми **New → Web Service**
3. Подключи свой GitHub репозиторий `battleship-online`
4. Заполни настройки:
   - **Name:** `battleship` (или любое)
   - **Region:** Frankfurt (ближе к Кипру)
   - **Branch:** `main`
   - **Runtime:** `Node`
   - **Build Command:** `npm install`
   - **Start Command:** `npm start`
   - **Instance Type:** `Free`
5. Нажми **Create Web Service**

Render сам соберёт и запустит проект. Через 2–3 минуты получишь URL типа:
```
https://battleship-xxxx.onrender.com
```

### Шаг 3. Играть

Отправляй эту ссылку друзьям, заходите оба, один создаёт комнату, второй вводит код — и в бой.

## ⚠️ Особенности бесплатного тарифа Render

- Сервис **засыпает через 15 минут** бездействия. Первый заход после паузы — ~1 минуту просыпается. Дальше всё быстро.
- Состояние игр хранится в памяти сервера. При перезапуске активные комнаты исчезают — это норма для бесплатки.
- Если хочешь, чтобы сервис не засыпал — платный план $7/мес или внешний "пингер" типа UptimeRobot раз в 10 минут.
